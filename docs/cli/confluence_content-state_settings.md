# confluence content-state settings

Show Cloud content-state settings for a space

## Synopsis

Show Cloud space settings for content states.

## Examples

confluence content-state settings ENG
  confluence content-state settings ENG --json

## Usage

```text
confluence content-state settings <space-key>
```

## Inherited Options
