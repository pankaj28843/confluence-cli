# confluence docs

Generate CLI reference documentation

## Synopsis

Generate CLI reference documentation from Cobra command metadata.

## Examples

confluence docs markdown --out docs/cli

## Usage

```text
confluence docs
```

## Commands

- `markdown` - Generate Markdown CLI reference files
