# confluence task reopen

Mark one Cloud content task incomplete

## Synopsis

Mark one Confluence Cloud content task incomplete.

## Examples

confluence task reopen 42
  confluence task reopen 42 --json

## Usage

```text
confluence task reopen <id>
```
